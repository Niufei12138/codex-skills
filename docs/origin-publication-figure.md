# Origin Publication Figure

Create journal-ready Origin figures with QA.

## Skill

- Name: `origin-publication-figure`
- Source: [`skills/origin-publication-figure/SKILL.md`](../skills/origin-publication-figure/SKILL.md)

## Example Prompt

```text
Use $origin-publication-figure to create a journal-ready Origin figure, save the .opju source, and export high-quality image files.
```

## Description

Create high-quality scientific and journal-ready figures in Origin/OriginPro using the local originlab MCP. Use when the user asks Codex to plot with Origin, make publication-quality or manuscript figures, polish scientific graphs, fit data for an Origin figure, export Origin figures, save .opju source files, or create reusable Origin figure outputs from CSV, Excel, pasted data, or existing Origin worksheets.

## Install

Copy `skills/origin-publication-figure` into your local Codex skills directory, for example `~/.codex/skills/origin-publication-figure`.
