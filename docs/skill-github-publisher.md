# Skill GitHub Publisher

Publish local Codex skills to GitHub.

## Skill

- Name: `skill-github-publisher`
- Source: [`skills/skill-github-publisher/SKILL.md`](../skills/skill-github-publisher/SKILL.md)

## Example Prompt

```text
Use $skill-github-publisher to publish or sync a local Codex skill to a GitHub repository.
```

## Description

Publish or sync local Codex skills to a GitHub repository. Use when the user asks to share a skill, publish a skill to GitHub, sync updated skills, create a skills repository, update README/index/docs for shared skills, or push local skill changes after creating or updating SKILL.md.

## Install

Copy `skills/skill-github-publisher` into your local Codex skills directory, for example `~/.codex/skills/skill-github-publisher`.
